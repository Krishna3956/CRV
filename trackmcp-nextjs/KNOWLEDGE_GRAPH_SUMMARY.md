## What is Google Knowledge Graph?

A rich information card that appears in Google Search showing key facts about your business/website.

**Example**:
```
When someone searches "Track MCP", they see:
┌─────────────────────────────────┐
│  Track MCP                      │
│  Model Context Protocol Directory│
│                                 │
│  Website: trackmcp.com          │
│  Founded: 2024                  │
│  Category: Developer Tools      │
│  Description: 10,000+ MCP tools │
│                                 │
│  [Visit Website] [Learn More]   │
└─────────────────────────────────┘
```
